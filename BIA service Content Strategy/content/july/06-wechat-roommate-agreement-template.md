### 📋 BIA室友合租协议

**1. 作息安排**

- 工作日几点关灯/安静？ ________
- 周末几点关灯/安静？ ________
- 早上闹钟时间和次数？ ________
- 是否接受对方在公共区域学习到很晚？ 是 / 否

**2. 整洁与清扫**

- 公共区域（厨房、客厅、卫生间）谁来打扫？
  □ 轮流（每_____轮一次）
  □ 谁弄脏谁打扫
  □ 请钟点工（费用平摊）
- 厨房使用后多久清理？ ________
- 垃圾谁倒？多久倒一次？ ________
- 对方的整洁标准你能接受的底线是？ ________

**3. 访客规则**

- 可以带朋友来家里吗？ 是 / 否
- 如果可以，需要提前多久通知？ ________
- 一周最多几次？ ________
- 可以留宿吗？ 是 / 否
- 对方（男/女朋友）留宿的规则？ ________

**4. 噪音**

- 可以在房间里外放音乐/视频吗？ 是 / 否
- 可以接受的噪音时段？ ________到________
- 打电话/开会是否需要去自己房间？ 是 / 否

**5. 费用分担**

- 房租：各付各的 / 均摊 / 按房间大小分 ________
- 水电网：均摊 / 按使用量 ________
- 公共用品（纸巾、清洁用品等）：均摊 / 轮流买 ________
- 食物：完全分开 / 共享部分（哪些？）________
- 家具：谁买的搬走时归谁 / 按比例折算 ________
- 用什么方式结算？Venmo / Zelle / 微信 / Splitwise ________
- 多久结算一次？每月 / 即时 ________

**6. 温度与空调**

- 空调设定温度范围：____°F 到 ____°F
- 晚上开空调吗？ 是 / 否
- 开窗通风偏好？ ________

**7. 宠物**

- 是否计划养宠物？ 是 / 否
- 对室友养宠物是否接受？ 是 / 否
- 如果有宠物，清洁责任和费用？ ________

**8. 沟通方式**

- 有问题怎么沟通？ 直接说 / 微信说 / 定期聊 ________

  □ 各自冷静24小时再谈
  □ 其他：________

**9. 退租**

- 如果一方想提前搬走，需要提前多久通知？ ________

- 押金怎么处理？ ________

### 怎么用这个模板？

1. **搬进去之前** — 两个人坐下来（或视频通话）逐条聊一遍
2. **不要敷衍** — 每一条都认真讨论，有分歧现在解决比以后好
3. **各保存一份** — 微信聊天记录/共享文档都行
4. **允许调整** — 住进去后如果发现某条不合理，随时重新商量
