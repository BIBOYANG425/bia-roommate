# USCBIA GTM Implementation Plan

> **For the executor:** This is an operational marketing plan, not a code plan. Each task is a concrete deliverable with a deadline, owner slot, and done criteria. Work through tasks sequentially within each phase. Tasks across phases can overlap where noted.

**Goal:** Launch USCBIA's community-led GTM strategy to reach 500+ questionnaire completions and 200+ successful matches for the Fall 2026 USC admissions cycle.

**Approach:** Six parallel workstreams executed in four phases (Foundation → Seed → Peak Push → Harvest), March through September 2026.

**Key constraint:** Lean budget (¥6,500–12,500 / ~$900–1,700 USD). Community-led and content-first. Paid is optional amplification only.
